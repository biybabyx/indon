## 📲 Cara Install

### Langkah 1 — Download Cloudstream
Belum punya Cloudstream? Download di sini:
👉 [Download Cloudstream](https://github.com/recloudstream/cloudstream/releases)

### Langkah 2 — Tambahkan Repository

1. Buka **Cloudstream**
2. Masuk ke **Pengaturan** → **Ekstensi**
3. Klik **Tambahkan Repositori**
4. Masukkan URL repo ini:

```
https://raw.githubusercontent.com/GITHUB_USERNAME/MyCloudRepo/builds/repo.json
```

5. Klik **Tambahkan**

### Langkah 3 — Install Plugin

1. Setelah repo ditambahkan, daftar plugin akan muncul
2. Pilih plugin yang ingin diinstall
3. Klik **Install** ✅

---

## ⚠️ Catatan Penting

- Jika tidak bisa menambahkan repo, coba gunakan **VPN**
- Ini karena ISP mungkin memblokir raw GitHub URL
- Plugin diupdate secara **otomatis** oleh Cloudstream

## 🐛 Lapor Bug

Kalau ada plugin yang error atau tidak bisa putar video:
1. Buka tab **Issues** di repo ini
2. Klik **New Issue**
3. Jelaskan masalahnya (plugin apa, error apa)
